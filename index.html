Sample Code: Calculate Compound Interest

```python def compound_interest(...): ... ``` 

# Compound Interest Calculation
def compound_interest(principal, rate, times_compounded, years):
    return principal * (1 + rate / times_compounded) ** (times_compounded * years)

# Example:
ci = compound_interest(10000, 0.05, 4, 5)
print(f"Compound Interest: ${ci:.2f}")
Here is a Python example to calculate compound interest, a key concept in finance.

Financial Formula: Present Value

The Present Value (PV) of a future cash flow is calculated as:

The present value is calculated as \( PV = \frac{FV}{(1 + r)^n} \), which is important in finance.

\[ PV = \frac{FV}{(1 + r)^n} \]

  • FV: Future Value
  • r: discount rate
  • n: number of periods
Use this formula to determine the current value of future cash flows discounted at rate r.
